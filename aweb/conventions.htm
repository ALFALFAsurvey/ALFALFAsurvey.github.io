ALFALFA Naming Conventions

The "designated" observer

The designated observer is the person responsible for carrying out the observations on a given day, regardless of whether that person is actually the one driving the telescope.

The "designated" scheduler

The designated scheduler is the person responsible for assigning the declination strip/pass to be observed on a given day.

The "designated" data file monitor

The designated data file monitor is the person responsible for verifying the existence of the raw fits files and performing their conversion to IDL .sav files.

Observing block designations

The block name is assigned according to the date when the observing block begins, in AST. If there is any confusion, ask the designated scheduler. Note that where there are two separate blocks on the same day, one is designated "a" for "am" and the other "p" for "pm". If there are more than 2 blocks, the designated scheduler will be creative...

Drift nomenclature and corresponding declinations

The master list contains the declination assignments for each declination strip and pass for the entire ALFALFA survey.
